
# inmigracion-usa

Proyecto React (Vite) con el componente `InmigracionUSA` que proporcionaste.

Rápido para arrancar:

```bash
npm install
npm run dev
```

El proyecto usa Vite y la CDN de Tailwind (`https://cdn.tailwindcss.com`) para evitar pasos extra de configuración.

**Nota:** `lucide-react` está incluido como dependencia; si prefieres eliminarlo y usar iconos por CDN, avísame y lo ajusto.
